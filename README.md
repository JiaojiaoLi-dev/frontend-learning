# frontend-learning

前端学习练习仓库

## 介绍

这里存放我在前端学习过程中写的小案例、练习页面、课堂作业等，用于记录学习进度、保存练习代码。

## 学习内容

- HTML基础结构与语义化标签
- CSS基础样式与布局（正在学习中）
- JavaScript（暂未开始学习）

## 项目结构

~~~bash
frontend-learning/
├── 01-tech-daily/       # 科技日报练习（HTML 小页面）
│   ├── img/             # 页面图片资源
│   └── index.html       # 练习页面主文件
└── README.md            # 项目说明文档
~~~

## 使用方式

克隆仓库到本地后打开对应案例文件夹，直接运行‘.html’文件，即可在浏览器中浏览效果。
